# Volt Node ⚡

This Space runs a Volt Core node.

## Deployment Instructions
1. Download `volt-linux` artifact from GitHub.
2. Rename it to `volt_core` (no extension).
3. Upload `Dockerfile`, `run.sh`, `README.md`, and `volt_core` here.
4. Check Logs for Playit Claim URL.
